Spot Birds At These Trails!

Now that you're familiar with Missouri birding culture it's time to head outside and explore! Below is an interactive Google map displaying three popular trails each at Central, Southwest, Southeast, Northwest and Northeast locations. Each trail has been put on the map based on how frequently people track bird sightings on the Cornell Lab's eBird app each time they visit.

By clicking on each bird icon on the map, you'll find the trails address (some will have driving directions if there is no exact pinpoint location), contact information, corresponding website to learn more, and its eBird list. In case you'd like to have this information readily available, each trail's corresponding information is below the map (all the same information you can find from the Google map) organized by regional locations. Enjoy the sights and sounds out there — and don't forget to have fun!
